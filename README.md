# Sistema de Análisis de Ventas — Python

Este proyecto es una **herramienta de análisis de ventas desarrollada en Python**, diseñada para ayudar a tiendas y negocios a entender sus datos, medir desempeño y apoyar decisiones comerciales usando KPI’s, análisis visuales y pronósticos.

---

## Qué hace este sistema

✔ Calcula KPI’s clave de negocio  
✔ Genera reportes de ventas  
✔ Analiza tendencias por producto, cliente y fechas  
✔ Genera gráficas automáticas  
✔ Realiza un pronóstico de ventas  
✔ Está empaquetado como ejecutable para Windows

---

## Funcionalidades Principales

### KPI’s del negocio
- Ventas totales  
- Costo total  
- Utilidad total  
- Margen de ganancia  

### Reportes
Reportes diarios y mensuales de ventas y utilidad.

### Análisis de tendencias
- Ventas por fecha
- Ventas por categoría
- Productos más vendidos

### Gráficas
Genera gráficas:
- Ventas por día
- Ventas por categoría
- Productos más vendidos

### Pronóstico de ventas
Pronóstico simple de ventas para los próximos días basado en datos históricos.
